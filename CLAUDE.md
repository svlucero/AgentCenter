# AgentCenter Homepage

Homepage for AgentCenter, an application for managing agents associated with tasks (repositories, GitHub issues, runners, etc.).

## Architecture
- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **Animations:** Framer Motion (optional)
- **Language:** TypeScript

## Key Commands
- `npm install`: Install dependencies
- `npm run dev`: Start development server
- `npm run build`: Build for production
- `npm run start`: Start production server
- `npm run lint`: Run linting

## Development Conventions
- Use Functional Components and Hooks.
- Follow Tailwind CSS best practices.
- Ensure responsive design.
- Use Lucide React for consistent iconography.
- Maintain a clean, professional aesthetic similar to modern AI tool homepages.
